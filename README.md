# PROJECT MANAGEMENT SYSTEM

Is a simple web app to managing work flow in a system.

Equipped with admin features.

Stack technologies :

    Node.js (express)
    ejs (template engine)
    Postgresql (database)
    NPM (others package)
    Bootstrap 4.0 (View)
